About xslbook-cover

This library includes match and named templates to display book titles, subtitles, and descriptions. This library takes into account free positioning of them.

Usage

This section explains how to import xslbook-cover and call its templates in your XSL application or library.

Import xslbook-cover

To import xslbook-cover.xsl into another XSL file, the following handworks are needed. Since xslbook-cover depends on some xslet libraries, they are needed to import, too.

  1. Add the namespace declarations of xslbook and xslutil in a XSL file.

    <xsl-stylesheet version="1.0"
      xmlns:xsl="http://www.w3.org/1999/XSL/Transform"
      xmlns:ut="https://github.com/xslet/2020/xslutil"
      xmlns:book="https://github.com/xslet/2020/xslbook"> <!-- This! -->
      ...

  2. Add a import element of xslbook-cover in the XSL file.

      ...
      <!-- dependency XSL files are already imported in xslbook-cover.xsl -->
      <xsl:import href="./xslbook-cover.xsl"/>
      ...
